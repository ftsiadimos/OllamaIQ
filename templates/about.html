ğŸ”¬ Testing Process

  1. Connection: Connect to your Ollama instance (local or remote) and fetch available models
  2. Selection: Choose which models you want to benchmark from the available list
  3. Execution: Each selected model runs through a standardized set of test prompts
  4. Evaluation: Responses are evaluated for correctness, quality, and timing
  5. Results: Comprehensive results show performance metrics and allow comparison between models

ğŸ“ˆ Understanding Results

  • Smartness %: Overall intelligence score based on reasoning and problem-solving tests
  • Code %: Accuracy of code generation tasks
  • Mean/Median: Average and middle-point response times (lower is faster)
  • Min/Max: Fastest and slowest individual responses
